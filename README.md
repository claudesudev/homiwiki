# 호미위키 풀버전 데모

## 구성

```
homiwiki/
├── index.html
├── docs/
│   ├── _index.json
│   └── 청라-커낼웨이/
│       └── doc.json
```

## 실행 방법

브라우저에서 index.html을 직접 열면 fetch 차단으로 작동 안 함. 로컬 서버 필요.

### 방법 1 — Python

```bash
cd homiwiki
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.

### 방법 2 — Node.js (npx)

```bash
cd homiwiki
npx serve
```

### 방법 3 — VSCode Live Server 확장

index.html 우클릭 → Open with Live Server.

## 적용된 변경 사항

1. **컬러 토큰 전환**: 인디고(#5E6AD2) → 호미 그린(#1DB954). 사이트 전체 25곳에 일괄 적용.
2. **브레드크럼 위치**: 상단(top: 68px) → 하단 탭바 위(bottom: 72px). 패널은 위로 펼쳐짐.
3. **문서 페이지 V1 디자인**: 청라 커낼웨이 11개 챕터, 그래픽 중심 마크업.
4. **인덱스 단순화**: `_index.json`에 청라 커낼웨이 하나만.
5. **홈 큐레이션**: 클로드 카드 제거.
6. **TOPICS_LIST 더미 정리**: 클로드/온라인 환불 target 제거.

## 디자인 시스템

- 액센트: `--primary-action: #1DB954` (호미 그린)
- 다크 액센트: `#4ADE80`
- 챕터 카드: `--bg-elevated` + 0.5px border + 8% shadow
- 카드 간격: 6px (호흡 좁게)
- 본문 글씨: 13.5px / line-height 1.7
- 인포박스: 좌측 정렬 + 제브라 (0.025 음영)

## 청라 커낼웨이 챕터 (11개)

1. 한눈에 보기 — 단지·수로·호수 지도 SVG
2. 5개 테마존 — 전통·레저·예술문화·생태·타워
3. 얼마나 큰가요? — 면적 비교 가로 막대
4. 음악분수쇼 — 분수 비주얼 + 시간표

8. 피크닉·음식 — OK/NO 5개 행
9. 드라마에서 본 그곳 — 더 글로리·닥터스 카드

11. 알아두면 좋은 것 — 주차·버스·반려견·자전거
